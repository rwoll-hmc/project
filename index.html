    > Prompt (greater prompt) is a domain specific language (DSL) for easily expressing and controlling cue placements in a theatrical script or prompt book.

    This project has been completed by Ross Wollman as coursework for Domain Specific Languages at Harvey Mudd College taught by Professor Ben Wiedermann.
